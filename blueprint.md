
# Project Blueprint

## Overview

A simple, modern, dark-themed website to display film canister development times in Rodinal 1+25.

## Project Structure

*   `index.html`: The main HTML file.
*   `style.css`: The CSS file for styling the website.
*   `main.js`: The JavaScript file for managing data and creating dynamic content.
*   `blueprint.md`: This file, which outlines the project and the development plan.

## Current Plan

1.  **Extract CSS:** Move all styling from `main.js` to a dedicated `style.css` file for better maintainability.
2.  **Modernize the Color Palette:** Introduce a new, vibrant color palette using `oklch` and CSS variables.
3.  **Improve Typography:** Select a new font from Google Fonts and create a clearer visual hierarchy.
4.  **Refine Layout & Add Texture:** Improve spacing, visual balance, and add a subtle noise texture to the background.
5.  **Enhance Interactivity:** Add more engaging hover effects and transitions to UI elements.

## Implemented Features

*   Basic HTML structure.
*   Modern dark theme with custom properties for colors.
*   A `<film-canister>` web component to display film data.
*   Dynamic rendering of film data from a JavaScript array.
*   The provided film data has been added to `main.js`.
